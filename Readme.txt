Клонирование проекта в папку NameProject:
git clone https://... nameProject

Добавить файлы в коммит:
git add .

Записать коммит:
git commit -m "Комментарий коммита"

Отправить коммит:
git push

Принять изменения:
git pull 

Авторство коммитов:
git config --global user.email " Ваш майл "
git config --global user.name " Ваше имя "
